# 🏥 MediTrack - Clinic Appointment Management System

A console-based Clinic Appointment Management System developed using **Java 21** and **Maven**.

This project demonstrates Core Java, Object-Oriented Programming, Java Collections, Exception Handling, Java 8 Streams, Design Patterns, File Handling, and Git/GitHub best practices.

---

# Features

## Doctor Management

- Add Doctor
- View Doctors
- Search Doctor
- Delete Doctor

## Patient Management

- Add Patient
- View Patients
- Search Patient
- Delete Patient

## Appointment Management

- Book Appointment
- View Appointments
- Confirm Appointment
- Complete Appointment
- Cancel Appointment
- Search Appointment by Doctor
- Search Appointment by Patient

## Billing

- Generate Bill
- View Bills
- Bill Summary

# How to Run

Clone Repository

```bash
git clone https://github.com/Atharva-wdev/MediTrack-Clinic-Appointment-Management-System.git
```

Open in IntelliJ.

Run

```
mvn clean compile
```

Run

```
Main.java
```
