# Setup Instructions

## Prerequisites

- Java 21
- Maven 3.9+
- IntelliJ IDEA Community Edition
- Git

---

## Clone Repository

```bash
git clone https://github.com/Atharva-wdev/MediTrack-Clinic-Appointment-Management-System.git
```

---

## Open Project

Open IntelliJ IDEA.

Select

File → Open

Choose the project folder.

---

## Build

```
mvn clean compile
```

Expected Output

```
BUILD SUCCESS
```

---

## Run

Execute

Main.java
